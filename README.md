# ML Security Project

AI 기반 소스코드 취약점 분석 프로젝트

## 프로젝트 목표

- GitHub 기반 취약 코드 수집
- CodeBERT 기반 취약점 탐지 모델 학습
- SQL Injection / XSS / Command Injection 탐지

## 사용 기술

- Python
- Transformers
- CodeBERT
- PyTorch
- Google Colab
